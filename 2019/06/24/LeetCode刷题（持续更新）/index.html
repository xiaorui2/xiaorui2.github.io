思路

题意是让一个32位的带符号的整数倒置过来。

就先取绝对值，然后初始化一个result = 0，把给定的数字n除以十，得到余数作为最低位，商作为新的数字n，result = result + 余数*10，这样每次得到的最低位就会随着循环不断提升数位，从而得到逆序的result。
